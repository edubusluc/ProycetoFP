PROYECTO LABORATORIO: Barclays Premier League Games Won 2010-16
El arhivo csv contiene:
	-Los diferentes equipos que han participado en la Premier desde 2010-2015
	-Los partidos que han ganado en cada una de las temporada, diferenciando entre Home y Away
	-Las modificaciones realizadas son la eliminación de columnas de la temporada 2016, hemos agregado una fecha que corresponde con la finalización de cada temporada, además de la media de goles por temporada de cada equipo

El proyecto contien:
	-El archivo csv
	-Dos módulos pyhton, uno para la definción de las funciones y otro para realizar la prueba de dichas funciones
